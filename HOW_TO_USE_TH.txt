=====================================================
วิธีใช้งานระบบทำนายลายมือตัวเลขไทย (Thai Handwriting ML V8.1)
=====================================================

โปรเจคนี้ประกอบด้วย 3 ส่วนหลักที่เชื่อมโยงกัน:
1. Backend (FastAPI - หัวใจ AI)
2. Frontend (Next.js - ส่วนประสานงานผู้ใช้)
3. Cloudinary (ที่เก็บข้อมูลรูปภาพถาวร)

-----------------------------------------------------
ขั้นตอนที่ 1: การเตรียมสภาพแวดล้อม (Local Setup)
-----------------------------------------------------
1. เปิด Terminal ใน VS Code และเข้าไปที่โฟลเดอร์โปรเจค
   คำสั่ง: `cd thai-handwriting-next`
2. เปิดใช้งาน Virtual Environment (venv):
   คำสั่ง: `.\.venv\Scripts\Activate.ps1`
3. ติดตั้ง Library (ถ้ายังไม่เคยทำ):
   คำสั่ง: `pip install -r backend/requirements.txt` และ `npm install`

-----------------------------------------------------
ขั้นตอนที่ 2: การรันระบบเพื่อทดสอบ (Running)
-----------------------------------------------------
ต้องรัน 2 Terminal พร้อมกัน:

ส่วนที่ 1: รัน Backend
- คำสั่ง: `python backend/main.py` (จะรันที่ Port 8000)

ส่วนที่ 2: รัน Frontend
- คำสั่ง: `npm run dev` (จะรันที่ Port 3000)
- เปิด Browser ไปที่: `http://localhost:3000`

-----------------------------------------------------
ขั้นตอนที่ 3: การเก็บข้อมูล (Dataset Collection)
-----------------------------------------------------
1. ไปที่หน้าเว็บ แถบ **"Dataset"**
2. วาดตัวเลข (๓๖-๔๐) และเลือก Label ให้ถูกต้อง
3. กด **"บันทึกตัวอย่าง"**:
   - หากรันบนเครื่อง: รูปจะบันทึกลง `dataset/[เลขไทย]/`
   - หากใช้งานออนไลน์: รูปจะอัปโหลดตรงไปที่ **Cloudinary**
4. ระบบจะล้างกระดานวาดให้อัตโนมัติหลังบันทึกสำเร็จ

-----------------------------------------------------
ขั้นตอนที่ 4: การเทรนโมเดลใหม่ (Training V8.1)
-----------------------------------------------------
เมื่อมีข้อมูลเพียงพอ (แนะนำ 200+ รูปต่อคลาสเพื่อความแม่นยำสูง):
1. ใน Terminal รันคำสั่ง: `python train_model.py`
2. ระบบจะทำการ Preprocessing (V8.1: Padding 8px, No Dilation) และเทรนโมเดล ExtraTrees
3. เมื่อเสร็จสิ้น จะได้ไฟล์ `models/current_model.pkl` และ `models/metrics.json`
4. **การนำไปใช้:** 
   - รีสตาร์ท Backend (Ctrl+C แล้วรันใหม่)
   - หรืออัปโหลดผ่านหน้า Admin บนเว็บเพื่อใช้ทันที

-----------------------------------------------------
ขั้นตอนที่ 5: การใช้งาน (Predict & Admin)
-----------------------------------------------------
- **หน้าหลัก:** วาดตัวเลขและกด "Predict" เพื่อดูผลทำนายและค่า Confidence Score
- **หน้า Admin:** ติดตามค่า Accuracy (V8.1 อยู่ที่ 96.36%) และอัปเดตโมเดลแบบ Real-time

-----------------------------------------------------
การแก้ไขปัญหาเบื้องต้น (Troubleshooting)
-----------------------------------------------------
1. **AI ไม่ตอบสนอง (ออนไลน์):** อาจเกิดจาก Render Cold Start ให้รอ 30-60 วินาทีแล้วลองใหม่
2. **รูปภาพไม่บันทึก:** ตรวจสอบ API Key ของ Cloudinary ในไฟล์ `.env.local`
3. **ทำนายไม่แม่นยำ:** ตรวจสอบว่ารูปใน `dataset/` มีความหลากหลายพอหรือไม่ และลองวาดตัวเลขให้มีขนาดพอดีกับช่อง
