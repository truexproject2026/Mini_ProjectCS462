=====================================================
คู่มือการติดตั้งระบบออนไลน์ (Vercel + Render + Cloudinary)
=====================================================

โปรเจคนี้ใช้โครงสร้างแบบ Hybrid Cloud เพื่อรองรับ Machine Learning:
1. Frontend (Next.js) -> ฝากที่ Vercel
2. Backend (FastAPI) -> ฝากที่ Render.com
3. Dataset Storage   -> ฝากที่ Cloudinary (เก็บรูปถาวร)

-----------------------------------------------------
ขั้นตอนที่ 1: ตั้งค่า Cloudinary (สำหรับเก็บรูป Dataset)
-----------------------------------------------------
1. สมัครสมาชิกที่ https://cloudinary.com/
2. ในหน้า Dashboard ก๊อปปี้ค่าดังนี้:
   - Cloud Name
   - API Key
   - API Secret (กดปุ่มรูปดวงตาเพื่อดู)
3. ค่าเหล่านี้จะถูกใช้เป็น Environment Variables ใน Vercel

-----------------------------------------------------
ขั้นตอนที่ 2: ตั้งค่า Backend ไปที่ Render.com
-----------------------------------------------------
1. สมัครสมาชิก Render.com และเชื่อมต่อ GitHub
2. กด "New" > "Web Service" เลือก Repository ของคุณ
3. ตั้งค่าดังนี้:
   - Name: (ตามต้องการ เช่น thai-handwriting-api)
   - Runtime: Python 3
   - Root Directory: thai-handwriting-next
   - Build Command: pip install -r backend/requirements.txt
   - Start Command: uvicorn backend.main:app --host 0.0.0.0 --port $PORT
4. กด Deploy และก๊อปปี้ URL ของ Backend มา (เช่น https://...onrender.com)

-----------------------------------------------------
ขั้นตอนที่ 3: ตั้งค่า Frontend ไปที่ Vercel
-----------------------------------------------------
1. เข้า Vercel และเลือก "Add New" > "Project" ดึงมาจาก GitHub
2. ในหน้าตั้งค่า ให้ใส่ Environment Variables ดังนี้:
   - BACKEND_URL = (URL จาก Render ในข้อ 4)
   - NEXT_PUBLIC_BACKEND_URL = (URL จาก Render เหมือนข้างบน)
   - CLOUDINARY_CLOUD_NAME = (ค่าจากขั้นตอนที่ 1)
   - CLOUDINARY_API_KEY = (ค่าจากขั้นตอนที่ 1)
   - CLOUDINARY_API_SECRET = (ค่าจากขั้นตอนที่ 1)
3. ตั้งค่า Output Directory เป็น: .next (ใน Settings > General)
4. กด Deploy

-----------------------------------------------------
ขั้นตอนที่ 4: การอัปเดตโมเดลแบบ Real-time
-----------------------------------------------------
- คุณสามารถอัปเดตโมเดลได้ทันทีผ่านหน้า Admin ในเว็บออนไลน์
- เลือกไฟล์ .pkl แล้วกด "Upload & Update"
- ระบบจะส่งไฟล์ตรงไปที่ Render และโหลดเข้าหน่วยความจำทันที
- หมายเหตุ: การอัปโหลดผ่านเว็บเป็นค่าชั่วคราว หากต้องการเปลี่ยนถาวรต้อง Commit ลง GitHub
